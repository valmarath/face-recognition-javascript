<h4> Prerequisites</h4>

- Node >= 20.12.2 and npm >= 10.7.0
- Package manager - NPM or YARN

<h4>Starting</h4>

```
git clone https://github.com/valmarath/face-recognition-login.git
npm install
```

<h5>Start Application</h5>

```
  //in root
  npm run start
```
